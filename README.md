# LP 作成テンプレート

- ペライチの LP を作成するための開発環境

| コード                            | ビルド |
| --------------------------------- | ------ |
| Pug, Scss, JavaScript, TypeScript | Parcel |

## 環境構築

- Node v18

### 依存パッケージのインストール

```
yarn
```

### サーバーの起動

```
yarn start
```

### デプロイ時の build

```
yarn build
```
